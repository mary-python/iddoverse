# IDDO-SDTM Tutorial Trial Inclusion Exclusion Criteria (TI) Domain

A fictional TI dataset for testing code and training

## Usage

``` r
TI_RPTESTB
```

## Format

### 

A data frame with 7 rows and 5 columns:

- STUDYID:

  Study Identifier

- DOMAIN:

  Domain Abbreviation

- IETESTCD:

  Incl/Excl Criterion Short Name

- IETEST:

  Inclusion/Exclusion Criterion

- IECAT:

  Inclusion/Exclusion Category
