# IDDO-SDTM Tutorial Disease Response and Clinical Classification (RS) Domain

A ficRSonal RS dataset for tesRSng code and training

## Usage

``` r
RS_RPTESTB
```

## Format

### 

A data frame with 4 rows and 28 columns:

- STUDYID:

  Study Identifier

- DOMAIN:

  Domain Abbreviation

- USUBJID:

  Unique Subject Identifier

- RSSEQ:

  Sequence Number

- RSTESTCD:

  Assessment Short Name

- RSTEST:

  Assessment Name

- RSCAT:

  Category for Assessment

- RSORRES:

  Result or Finding in Original Units

- RSORRESU:

  Original Units

- RSSTRESC:

  Character Result/Finding in Std Format

- RSSTRESN:

  Numeric Result/Finding in Std Format

- RSSTRESU:

  Standard Units

- RSSTAT:

  Completion Status

- RSREASND:

  Reason Not Done

- VISITNUM:

  Visit Number

- VISIT:

  Visit Name

- VISITDY:

  Planned Study Day of Visit

- EPOCH:

  Epoch

- RSDTC:

  Date/Time of Assessment

- RSDY:

  Study Day of Assessment

- RSTPT:

  Planned Time Point Name

- RSTPTNUM:

  Planned Time Point Number

- RSTPTREF:

  Time Point Reference

- RSSTRF:

  Start Relative to Reference Period

- RSEVINTX:

  Evaluation Interval Text

- RSSTRTPT:

  Start Relative to Reference Time Point

- RSSTTPT:

  Start Reference Time Point

- RSCDSTDY:

  Contributor-Defined Study Day
