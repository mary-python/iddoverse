# IDDO-SDTM Tutorial Pharmacogenomics Genetics (PF) Domain

A fictional PF dataset for testing code and training

## Usage

``` r
PF_RPTESTB
```

## Format

### 

A data frame with 12 rows and 26 columns:

- STUDYID:

  Study Identifier

- DOMAIN:

  Domain Abbreviation

- USUBJID:

  Unique Subject Identifier

- NHOID:

  Non-Host Organism Identifier

- PFSEQ:

  Sequence Number

- PFREFID:

  Reference ID

- PFTESTCD:

  Short Name of Pharmacogenomics Lab Test

- PFTEST:

  Name of Pharmacogenomics Lab Test

- PFGENRI:

  Genetic Region of Interest

- PFGENTRG:

  Genetic Target

- PFCAT:

  Category for Pharmacogenomics Test

- PFORRES:

  Result or Finding in Original Units

- PFORRESU:

  Original Units

- PFSTRESC:

  Result or Finding in Standard Format

- PFSTRESN:

  Numeric Result/Finding in Standard Units

- PFSTRESU:

  Standard Units

- PFSPEC:

  Specimen Type

- PFMETHOD:

  Method Code for Test

- VISITNUM:

  Visit Number

- VISIT:

  Visit Name

- VISITDY:

  Planned Study Day of Visit

- EPOCH:

  Epoch

- PFDTC:

  Date/Time of Specimen Collection

- PFDY:

  Study Day of Specimen Collection

- PFCDSTDY:

  Contributor-Defined Study Day
