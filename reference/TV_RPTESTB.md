# IDDO-SDTM Tutorial Trial Visits (TV) Domain

A fictional TV dataset for testing code and training

## Usage

``` r
TV_RPTESTB
```

## Format

### 

A data frame with 20 rows and 8 columns:

- STUDYID:

  Study Identifier

- DOMAIN:

  Domain Abbreviation

- VISITNUM:

  Visit Number

- VISIT:

  Visit Name

- VISITDY:

  Planned Study Day of Visit

- ARMCD:

  Planned Arm Code

- ARM:

  Description of Planned Arm

- TVSTRL:

  Visit Start Rule
