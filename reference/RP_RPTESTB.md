# IDDO-SDTM Tutorial Reproductive System Findings (RP) Domain

A fictional RP dataset for testing code and training

## Usage

``` r
RP_RPTESTB
```

## Format

### 

A data frame with 1 row and 18 columns:

- STUDYID:

  Study Identifier

- DOMAIN:

  Domain Abbreviation

- RPSEQ:

  Sequence Number

- USUBJID:

  Unique Subject Identifier

- RPTESTCD:

  Short Name of Reproductive Test

- RPTEST:

  Name of Reproductive Test

- RPORRES:

  Result or Finding in Original Units

- RPORRESU:

  Original Units

- RPMODIFY:

  Pre-Specified Event

- RPSTRESC:

  Character Result/Finding in Std Format

- RPSTRESN:

  Numeric Result/Finding in Standard Units

- RPSTRESU:

  Standard Units

- VISITNUM:

  Visit Number

- VISIT:

  Visit Name

- VISITDY:

  Planned Study Day of Visit

- EPOCH:

  Epoch

- RPDTC:

  Date/Time of Collection

- RPDY:

  Study Day of Visit/Collection/Exam
