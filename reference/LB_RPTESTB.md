# IDDO-SDTM Tutorial Laboratory Test Results (LB) Domain

A fictional LB dataset for testing code and training

## Usage

``` r
LB_RPTESTB
```

## Format

### 

A data frame with 10 rows and 19 columns:

- STUDYID:

  Study Identifier

- DOMAIN:

  Domain Abbreviation

- USUBJID:

  Unique Subject Identifier

- LBSEQ:

  Sequence Number

- LBTESTCD:

  Lab Test or Examination Short Name

- LBTEST:

  Lab Test or Examination Name

- LBORRES:

  Result or Finding in Original Units

- LBMODIFY:

- LBORRESU:

  Original Units

- LBSTRESC:

  Character Result/Finding in Std Format

- LBSTRESN:

  Numeric Result/Finding in Standard Units

- LBSTRESU:

  Standard Units

- VISITNUM:

  Visit Number

- VISIT:

  Visit Name

- VISITDY:

  Planned Study Day of Visit

- EPOCH:

  Epoch

- LBDTC:

  Date/Time of Specimen Collection

- LBDY:

  Study Day of Specimen Collection

- LBEVINTX:

  Evaluation Interval Text
