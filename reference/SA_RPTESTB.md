# IDDO-SDTM Tutorial Clinical and Adverse Events (SA) Domain

A fictional SA dataset for testing code and training

## Usage

``` r
SA_RPTESTB
```

## Format

### 

A data frame with 3 rows and 21 columns:

- STUDYID:

  Study Identifier

- DOMAIN:

  Domain Abbreviation

- SASEQ:

  Sequence Number

- USUBJID:

  Unique Subject Identifier

- SATERM:

  Reported Term for the Event

- SAMODIFY:

  Modified Reported Term

- SADECOD:

  Dictionary-Derived Term

- SACAT:

  Category for Event

- SAPRESP:

  Pre-Specified Event

- SAOCCUR:

  Clinical Event Occurrence

- SASEV:

  Severity/Intensity

- VISITNUM:

  Visit Number

- VISIT:

  Visit Name

- VISITDY:

  Planned Study Day of Visit

- EPOCH:

  Epoch

- SADY:

  Study Day of Specimen Collection

- SADTC:

  Date/Time of Specimen Collection

- SASTDY:

  Study Day of Start of Event

- SAENDY:

  Study Day of End of Event

- SADUR:

  Duration

- SAEVINTX:

  Evaluation Interval Text
