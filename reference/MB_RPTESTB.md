# IDDO-SDTM Tutorial Microbiology (MB) Domain

A fictional MB dataset for testing code and training

## Usage

``` r
MB_RPTESTB
```

## Format

### 

A data frame with 14 rows and 39 columns:

- STUDYID:

  Study Identifier

- DOMAIN:

  Domain Abbreviation

- USUBJID:

  Unique Subject Identifier

- MBSEQ:

  Sequence Number

- MBGRPID:

  Group ID

- MBREFID:

  Reference ID

- MBTESTCD:

  Microbiology Test or Finding Short Name

- MBTEST:

  Microbiology Test or Finding Name

- MBMODIFY:

  Modified Term

- MBTSTDTL:

  Measurement, Test or Examination Detail

- MBCAT:

  Category

- MBORRES:

  Result or Finding in Original Units

- MBORRESU:

  Original Units

- MBSTRESC:

  Result or Finding in Standard Format

- MBSTRESN:

  Numeric Result/ Finding in Standard Units

- MBSTRESU:

  Standard Units

- MBSTAT:

  Completion Status

- MBREASND:

  Reason Not Done

- MBNAM:

  Laboratory/ Vendor Name

- MBSPEC:

  Specimen Material Type

- MBSPCCND:

  Specimen Condition

- MBLOC:

  Specimen Collection Location

- MBMETHOD:

  Method of Test or Examination

- VISITNUM:

  Visit Number

- VISIT:

  Visit Name

- VISITDY:

  Planned Study Day of Visit

- EPOCH:

  Epoch

- MBDTC:

  Date/ Time of Collection

- MBDY:

  Study Day of Visit/ Collection/ Exam

- MBTPT:

  Planned Time Point Name

- MBTPTNUM:

  Planned Time Point Number

- MBELTM:

  Planned Elapsed Time from Time Point Ref

- MBTPTREF:

  Time Point Reference

- MBSTRF:

  Start Relative to Reference Period

- MBEVINTX:

  Evaluation Interval Text

- MBSTRTPT:

  Start Relative to Reference Time Point

- MBSTTPT:

  Start Reference Time Point

- MBCDSTDY:

  Contributor-Defined Study Day

- MBRPOC:

  Reported Observation Occurrence
