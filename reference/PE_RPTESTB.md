# IDDO-SDTM Tutorial Physical Examination (PE) Domain

A fictional PE dataset for testing code and training

## Usage

``` r
PE_RPTESTB
```

## Format

### 

A data frame with 12 rows and 22 columns:

- STUDYID:

  Study Identifier

- DOMAIN:

  Domain Abbreviation

- USUBJID:

  Unique Subject Identifier

- PESEQ:

  Sequence Number

- PETESTCD:

  Body System Examined Short Name

- PETEST:

  Body System Examined

- PECAT:

  Category for Examination

- PESCAT:

  Subcategory for Examination

- PEBODSYS:

  Body System or Organ Class

- PEORRES:

  Verbatim Examination Finding

- PEORRESU:

  Original Units

- PESTRESC:

  Character Result/ Finding in Std Format

- PESTAT:

  Completion Status

- PEREASND:

  Reason Not Examined

- PELOC:

  Location of Physical Exam Finding

- PEMETHOD:

  Method of Test or Examination

- VISITNUM:

  Visit Number

- VISIT:

  Visit Name

- VISITDY:

  Planned Study Day of Visit

- EPOCH:

  Epoch

- PEDTC:

  Date/Time of Examination

- PEDY:

  Study Day of Examination
