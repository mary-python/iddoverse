# IDDO-SDTM Tutorial Disposition (DS) Domain

A fictional DS dataset for testing code and training

## Usage

``` r
DS_RPTESTB
```

## Format

### 

A data frame with 3 rows and 16 columns:

- STUDYID:

  Study Identifier

- DOMAIN:

  Domain Abbreviation

- USUBJID:

  Unique Subject Identifier

- DSSEQ:

  Sequence Number

- DSTERM:

  Reported Term for the Disposition Event

- DSMODIFY:

  Modified Reported Term

- DSDECOD:

  StandardiSed Disposition Term

- VISITNUM:

  Visit Number

- VISIT:

  Visit Name

- VISITDY:

  Planned Study Day of Visit

- EPOCH:

  Epoch

- DSDTC:

  Date/Time of Specimen Collection

- DSSTDTC:

  Start Date/Time of Disposition Event

- DSDY:

  Study Day of Specimen Collection

- DSSTDY:

  Study Day of Start of Disposition Event

- DSCDSTDY:

  Contributor-Defined Study Day
